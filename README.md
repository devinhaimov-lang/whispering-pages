# Whispering Pages

## Project Description
Whispering Pages is an innovative application that allows users to delve into their favorite stories in a unique way. You can interact with book characters through a chat interface, bringing literature to life like never before.

## Features
- Chat with book characters
- Explore various stories and genres
- Bilingual support (English & Chinese)

## Tech Stack
- JavaScript
- React
- Node.js
- Express
- MongoDB

## Getting Started
To get started with Whispering Pages, clone the repository and install the required dependencies:

```bash
git clone https://github.com/devinhaimov-lang/whispering-pages.git
cd whispering-pages
npm install
```

### Special Feature
Our standout feature is the ability to chat with book characters, making your reading experience interactive and immersive!
