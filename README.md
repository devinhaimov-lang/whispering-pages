# Whispering Pages

Whispering Pages is a book reading application developed using **React**, **TypeScript**, and **Vite**. This application aims to provide a seamless and engaging reading experience for book enthusiasts.

## Features

- **Library**: Browse and manage your collection of books with ease.
- **Book Reader**: Enjoy reading books in a user-friendly interface that enhances the reading experience.
- **Admin Dashboard**: Manage users, books, and other application settings through a dedicated admin panel.
- **Supabase Integration**: Store and retrieve data efficiently with Supabase, providing backend support for your reading adventure.

## Getting Started

Follow the instructions here to set up the project locally.

1. Clone the repository:
   ```bash
   git clone https://github.com/devinhaimov-lang/whispering-pages.git
   cd whispering-pages
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

Enjoy reading!
